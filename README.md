# Proyecto Final para Desarrollo de Aplicaciones Web en 2026

Página con una lista de los juegos principales de Kingdom Hearts que incluye el logo, fecha de publicación del juego original y su verión remasterizada o ampliación (si aplicable), y el opening del juego detrás de una pestaña que tienes que pulsar para abrir.

## Ejemplo de como verlo

Descargarte este repositorio y la aplicación [LocalWP](https://localwp.com/https://localwp.com/), crea una página de WordPress, pulsa en el icono de la carpeta en la página creada, entra en "app" y "public", borra todo lo que hay dentro (por lo que no se usa para WordPress en este caso) y mete la copia de este repositorio dentro, finalmente, inicia la página y dale botón de abrir la página, este sería un método para poder ver el contenido de este repositorio.
